## 🚀 How to Run the Project Locally

### 1. Install Dependencies
Make sure you have Node.js installed, then run the package installer in the project root:
```bash
npm install
```

### 2. Run the Development Server
Launch the local Vite server:
```bash
npm run dev
```
Open `http://localhost:5173` in your web browser to view the live landing page.

### 3. Build for Production
To bundle and compile the application for deployment:
```bash
npm run build

```
